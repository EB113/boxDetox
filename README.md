#Setup guide# <br/>
./setup <br/>

./boxdetox - open regular program <br/>
./boxdetox logger - opens another program, Modules log here by default if this is detected when main program launches.<br/>
or <br/>
apt install tmux <br/>
./start {{proj_name}} - open default tmux setup <br/>
